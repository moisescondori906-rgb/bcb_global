# Carpeta de Aplicaciones BCB Global

Coloque aquí el archivo APK para Android con el nombre:
**BCB_Global.apk**

Este archivo será servido automáticamente por la plataforma cuando un usuario presione "Descargar APK" desde un dispositivo Android.

---

### Notas para iPhone:
Para usuarios de iPhone, el sistema redirigirá automáticamente a la App de Google en el App Store según lo solicitado.
